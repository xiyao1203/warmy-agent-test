## 验收条件

- 登录、退出、当前用户和服务端 Session 安全测试通过。
- 仅超级管理员可以管理用户，且不能禁用或删除最后一个可用超级管理员。
- 普通用户只能访问被分配项目，跨项目直接访问返回 404。
- 用户禁用、密码重置或移出项目后权限立即失效。
- 身份、项目和审计数据库迁移及约束验证通过。
- 前端登录、用户管理、项目切换和权限状态通过组件与 E2E 验证。
